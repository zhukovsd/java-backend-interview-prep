#### 30. ForkJoinPool vs FixedThreadPool


| –°–≤–æ–π—Å—Ç–≤–æ                     | `ForkJoinPool`                               | `FixedThreadPool`                    |
| ---------------------------- | -------------------------------------------- | ------------------------------------ |
| **–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å?**      | –†–∞–∑–¥–µ–ª–µ–Ω–∏–µ –∑–∞–¥–∞—á (recursion)                 | –§–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ —á–∏—Å–ª–æ –ø–æ—Ç–æ–∫–æ–≤          |
| **–†–∞–∑–º–µ—Ä –ø—É–ª–∞**              | –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç—Å—è                  | –§–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ —á–∏—Å–ª–æ –ø–æ—Ç–æ–∫–æ–≤          |

| **–¢–∏–ø –∑–∞–¥–∞—á**                | –†–∞–∑–¥–µ–ª—è–µ–º—ã–µ (`RecursiveTask`)                | –ù–µ–∑–∞–≤–∏—Å–∏–º—ã–µ (`Runnable`, `Callable`) |

**–ü—Ä–∏–º–µ—Ä `ForkJoinPool` (—Ä–∞–∑–±–∏–µ–Ω–∏–µ –∑–∞–¥–∞—á–∏)**

```java
import java.util.concurrent.*;

class SumTask extends RecursiveTask<Integer> {
    private final int[] array;
    private final int start, end;

    SumTask(int[] array, int start, int end) {
        this.array = array;
        this.start = start;
        this.end = end;
    }

    @Override
    protected Integer compute() {
        if (end - start <= 2) {
            return array[start] + array[end - 1]; // –ë–∞–∑–æ–≤—ã–π —Å–ª—É—á–∞–π
        }

        int mid = (start + end) / 2;
        SumTask leftTask = new SumTask(array, start, mid);
        SumTask rightTask = new SumTask(array, mid, end);

        leftTask.fork();
        return rightTask.compute() + leftTask.join();
    }
}

public class ForkJoinExample {
    public static void main(String[] args) {
        ForkJoinPool pool = new ForkJoinPool();
        int[] numbers = {1, 2, 3, 4, 5, 6, 7, 8};

        int sum = pool.invoke(new SumTask(numbers, 0, numbers.length));
        System.out.println("–°—É–º–º–∞: " + sum); // 36
    }
}
```

‚úÖ **–ü–ª—é—Å—ã `ForkJoinPool`**  
‚úî **–†–∞–∑–±–∏–≤–∞–µ—Ç –∑–∞–¥–∞—á–∏ –Ω–∞ –ø–æ–¥–∑–∞–¥–∞—á–∏** (divide-and-conquer).  
‚úî **Work-stealing** —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å.

‚ùå **–ú–∏–Ω—É—Å—ã `ForkJoinPool`**  
‚õî –ù–µ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è **–Ω–µ–∑–∞–≤–∏—Å–∏–º—ã—Ö –∑–∞–¥–∞—á** (–ª—É—á—à–µ `FixedThreadPool`).

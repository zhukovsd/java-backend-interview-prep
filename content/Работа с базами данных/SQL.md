#### 3. –ü—Ä–æ–±–ª–µ–º—ã –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω—ã—Ö —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π

#### 21. –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç Serializable?	

`Serializable` ‚Äî —ç—Ç–æ —Å–∞–º—ã–π —Å—Ç—Ä–æ–≥–∏–π —É—Ä–æ–≤–µ–Ω—å –∏–∑–æ–ª—è—Ü–∏–∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π –≤ –±–∞–∑–∞—Ö –¥–∞–Ω–Ω—ã—Ö.

#### 23. C –ø–æ–º–æ—â—å—é SQL –±–ª–æ–∫–∏—Ä–æ–≤–∫–∞ —Å—Ç—Ä–æ–∫–∏? - SELECT FOR UPDATE	

**`SELECT FOR UPDATE`** –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –±–ª–æ–∫–∏—Ä–æ–≤–∫–∏ —Å—Ç—Ä–æ–∫ –≤ —Ç–∞–±–ª–∏—Ü–µ, —á—Ç–æ–±—ã –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—Ç–∏—Ç—å –∏—Ö –∏–∑–º–µ–Ω–µ–Ω–∏–µ –¥—Ä—É–≥–∏–º–∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º–∏ –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ —Ç–µ–∫—É—â–∞—è —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è –Ω–µ –∑–∞–≤–µ—Ä—à–∏—Ç—Å—è (—á–µ—Ä–µ–∑ `COMMIT` –∏–ª–∏ `ROLLBACK`).


```sql

-- –ë–ª–æ–∫–∏—Ä–æ–≤–∫–∞ —Å—Ç—Ä–æ–∫–∏ —Å id = 1

WHERE id = 1
```

- –ï—Å–ª–∏ –¥—Ä—É–≥–∞—è —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è –ø–æ–ø—ã—Ç–∞–µ—Ç—Å—è –∑–∞–±–ª–æ–∫–∏—Ä–æ–≤–∞—Ç—å —Ç—É –∂–µ —Å—Ç—Ä–æ–∫—É, –æ–Ω–∞ –±—É–¥–µ—Ç –æ–∂–∏–¥–∞—Ç—å –∑–∞–≤–µ—Ä—à–µ–Ω–∏—è —Ç–µ–∫—É—â–µ–π —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–∏.

#### 24. UNION vs UNION ALL

#### 26. –ß—Ç–æ —Ç–∞–∫–æ–µ —Ç–∞–±–ª–∏—á–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è?

‚õî **–ë–µ–∑ –∏–Ω–¥–µ–∫—Å–∞:** `O(N)` (–º–µ–¥–ª–µ–Ω–Ω–æ).  
‚úÖ **–° PK (–∏–Ω–¥–µ–∫—Å–æ–º):** `O(log N)` (–Ω–∞–º–Ω–æ–≥–æ –±—ã—Å—Ç—Ä–µ–µ).


 **‚ùå 3. –ü—Ä–æ–±–ª–µ–º—ã —Å –≤–Ω–µ—à–Ω–∏–º–∏ –∫–ª—é—á–∞–º–∏ (FOREIGN KEY)**

 **‚ùå 4. –£—Å–ª–æ–∂–Ω–µ–Ω–Ω–∞—è —Ä–µ–ø–ª–∏–∫–∞—Ü–∏—è –≤ MySQL**

#### 31. Foreign Key

`FOREIGN KEY` ‚Äì —ç—Ç–æ **–æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ** (`constraint`), –∫–æ—Ç–æ—Ä–æ–µ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç —Å—Å—ã–ª–æ—á–Ω—É—é —Ü–µ–ª–æ—Å—Ç–Ω–æ—Å—Ç—å –º–µ–∂–¥—É —Ç–∞–±–ª–∏—Ü–∞–º–∏. –û–Ω —É–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ **–∑–Ω–∞—á–µ–Ω–∏–µ –≤ –æ–¥–Ω–æ–º —Å—Ç–æ–ª–±—Ü–µ –¥–æ–ª–∂–Ω–æ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—é –≤ –¥—Ä—É–≥–æ–π —Ç–∞–±–ª–∏—Ü–µ (–æ–±—ã—á–Ω–æ Primary Key)**.

üìå **–ü—Ä–∏–º–µ—Ä:**

‚úÖ –¢–µ–ø–µ—Ä—å `orders.user_id` **–º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Ç–æ–ª—å–∫–æ —Ç–µ –∑–Ω–∞—á–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –≤ `users.user_id`**.

---
***Foreign Key –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–æ—Å—Ç–∞–≤–Ω—ã–º?***

‚úÖ **–î–∞, Foreign Key –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–æ—Å—Ç–∞–≤–Ω—ã–º (Composite Foreign Key).**

üìå **–ü—Ä–∏–º–µ—Ä:**

```sql

    category_id INT,
    subcategory_id INT,
    PRIMARY KEY (category_id, subcategory_id)

    product_id INT PRIMARY KEY,
    category_id INT,
    subcategory_id INT,

```

üîπ –¢–µ–ø–µ—Ä—å `products.category_id, subcategory_id` **–¥–æ–ª–∂–Ω—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞—Ç—å** `categories.category_id, subcategory_id`.

‚ùå **–ú–∏–Ω—É—Å—ã —Å–æ—Å—Ç–∞–≤–Ω–æ–≥–æ FK:**  
‚õî –£—Å–ª–æ–∂–Ω—è–µ—Ç –∏–Ω–¥–µ–∫—Å–∞—Ü–∏—é.  
‚õî –ó–∞–ø—Ä–æ—Å—ã `JOIN` —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è –º–µ–Ω–µ–µ —É–¥–æ–±–Ω—ã–º–∏.

|**–•–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞**|**Primary Key (PK)**|**Foreign Key (FK)**|

‚úÖ –ï—Å–ª–∏ —É–¥–∞–ª–∏—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è `users.user_id = 1`, —Ç–æ **–≤—Å–µ –µ–≥–æ –∑–∞–∫–∞–∑—ã —É–¥–∞–ª—è—Ç—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏!**

#### 32. DELETE vs TRUNCATE

`DELETE` –∏ `TRUNCATE` –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è —É–¥–∞–ª–µ–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –∏–∑ —Ç–∞–±–ª–∏—Ü—ã, –Ω–æ —Ä–∞–±–æ—Ç–∞—é—Ç **–ø–æ-—Ä–∞–∑–Ω–æ–º—É**.


 **1Ô∏è‚É£ DELETE**

‚úÖ **–ü–ª—é—Å—ã:**

- –£–¥–∞–ª—è–µ—Ç —Å—Ç—Ä–æ–∫–∏ –∏–∑ —Ç–∞–±–ª–∏—Ü—ã —Å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å—é —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏ (`WHERE`).
- –§–∏–∫—Å–∏—Ä—É–µ—Ç—Å—è –≤ –∂—É—Ä–Ω–∞–ª–µ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π (–ª–æ–≥–∏—Ä—É–µ—Ç—Å—è).
- –ú–æ–∂–Ω–æ –æ—Ç–∫–∞—Ç–∏—Ç—å (`ROLLBACK`), –µ—Å–ª–∏ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è `TRANSACTION`.
- –ó–∞–ø—É—Å–∫–∞–µ—Ç —Ç—Ä–∏–≥–≥–µ—Ä—ã (`AFTER DELETE`).

üìå **–ü—Ä–∏–º–µ—Ä:** –£–¥–∞–ª–µ–Ω–∏–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π —Å—Ç–∞—Ä—à–µ 30 –ª–µ—Ç

```sql

```

```sql

```

‚ùå **–ú–∏–Ω—É—Å—ã:**

- –ú–µ–¥–ª–µ–Ω–Ω—ã–π –¥–ª—è –±–æ–ª—å—à–∏—Ö —Ç–∞–±–ª–∏—Ü (—É–¥–∞–ª—è–µ—Ç —Å—Ç—Ä–æ–∫–∏ –ø–æ –æ–¥–Ω–æ–π).
- –¢—Ä–µ–±—É–µ—Ç –º–Ω–æ–≥–æ –ø–∞–º—è—Ç–∏ –∏–∑-–∑–∞ –ª–æ–≥–∏—Ä–æ–≤–∞–Ω–∏—è.

 **2Ô∏è‚É£ TRUNCATE**

‚úÖ **–ü–ª—é—Å—ã:**

- –£–¥–∞–ª—è–µ—Ç –≤—Å–µ —Å—Ç—Ä–æ–∫–∏ —Å—Ä–∞–∑—É (–±–µ–∑ `WHERE`).
- –ù–µ –ª–æ–≥–∏—Ä—É–µ—Ç –∫–∞–∂–¥—É—é —Å—Ç—Ä–æ–∫—É (—Ä–∞–±–æ—Ç–∞–µ—Ç –±—ã—Å—Ç—Ä–µ–µ).

- –ü–æ–ª–Ω–æ—Å—Ç—å—é –æ—á–∏—â–∞–µ—Ç —Ç–∞–±–ª–∏—Ü—É.

üìå **–ü—Ä–∏–º–µ—Ä:**


üí° **–†–∞–±–æ—Ç–∞–µ—Ç –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º `DELETE`**, —Ç–∞–∫ –∫–∞–∫ —É–¥–∞–ª—è–µ—Ç –≤—Å—é —Ç–∞–±–ª–∏—Ü—É —Å—Ä–∞–∑—É.

‚ùå **–ú–∏–Ω—É—Å—ã:**

- –ù–µ–ª—å–∑—è —É–¥–∞–ª–∏—Ç—å —Ç–æ–ª—å–∫–æ —á–∞—Å—Ç—å —Å—Ç—Ä–æ–∫ (`WHERE` –Ω–µ —Ä–∞–±–æ—Ç–∞–µ—Ç).
- –ù–µ–ª—å–∑—è –æ—Ç–∫–∞—Ç–∏—Ç—å (`ROLLBACK`) –≤ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ –°–£–ë–î.
- –ù–µ –≤—ã–∑—ã–≤–∞–µ—Ç `DELETE`-—Ç—Ä–∏–≥–≥–µ—Ä—ã.
- –ù–µ —Ä–∞–±–æ—Ç–∞–µ—Ç, –µ—Å–ª–∏ –µ—Å—Ç—å Foreign Key —Å `ON DELETE CASCADE`.

 **DELETE vs TRUNCATE: –°—Ä–∞–≤–Ω–µ–Ω–∏–µ**

|**–§—É–Ω–∫—Ü–∏—è**|**DELETE**|**TRUNCATE**|
|---|---|---|
|**–£–¥–∞–ª—è–µ—Ç –¥–∞–Ω–Ω—ã–µ?**|‚úÖ –î–∞, –º–æ–∂–Ω–æ –≤—ã–±—Ä–∞—Ç—å —Å—Ç—Ä–æ–∫–∏ (`WHERE`)|‚úÖ –î–∞, –Ω–æ —É–¥–∞–ª—è–µ—Ç –≤—Å—ë|

|**–ë—ã—Å—Ç—Ä–æ–¥–µ–π—Å—Ç–≤–∏–µ**|‚ùå –ú–µ–¥–ª–µ–Ω–Ω–µ–µ (–ª–æ–≥–∏—Ä—É–µ—Ç –∫–∞–∂–¥—É—é —Å—Ç—Ä–æ–∫—É)|‚úÖ –ë—ã—Å—Ç—Ä–æ (–Ω–µ –ª–æ–≥–∏—Ä—É–µ—Ç)|
|**–°–±—Ä–∞—Å—ã–≤–∞–µ—Ç AUTO_INCREMENT?**|‚ùå –ù–µ—Ç|‚úÖ –î–∞|
|**–í—ã–∑–æ–≤ —Ç—Ä–∏–≥–≥–µ—Ä–æ–≤ (`AFTER DELETE`)?**|‚úÖ –î–∞|‚ùå –ù–µ—Ç|
|**–ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Å FOREIGN KEY?**|‚úÖ –î–∞|‚ùå –ù–µ—Ç, –µ—Å–ª–∏ –µ—Å—Ç—å FK|
